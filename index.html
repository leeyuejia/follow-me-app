ABOUT DEV

The developer's motivation for this game is to assist doctor to assess demented patient's memory capabilities. Doctors can use this game during each patient's routine review to diagnose the rate of degeneration. Demented patient can also use this game to exercise memory muscles to slow the rate of degeneration. Feel free to reach out to the developer via the below media.
